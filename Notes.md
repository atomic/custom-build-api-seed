Using curl for get/post/put
=====================

# Get
```
    curl --requests GET "http://..."
    curl --request POST 'http://www.somedomain.com/'
    curl --request DELETE 'http://www.somedomain.com/'
    curl --request PUT 'http://www.somedomain.com/'
```
